# Be2

A fully localized, privacy-focused customer service and assistant bot named Be2.
It operates entirely off a local Ollama instance, ensuring no data ever leaves your servers.

## Features
- Connects to local Ollama (`qwen3:4b` by default or any other model)
- Provides a Telegram Bot interface
- Provides an API for Web Integration

## Setup
1. Copy `.env.example` to `.env` and configure your Telegram bot token.
2. Run `npm install`
3. Run `npm start`
